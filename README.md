## Features

- Accept payments using **Stripe**  
- Built with **Flask backend API**  
- Responsive frontend with **React + Vite**  
- Secure handling of API keys  
- Demo pages for testing payments  
- Ready for deployment online  

---

## Tech Stack

- **Backend:** Python, Flask, Stripe API  
- **Frontend:** React, Vite, Tailwind CSS (optional)  
- **Database:** SQLite/PostgreSQL (optional for orders)  
- **Version Control:** Git + GitHub  
- **Deployment Options:** Render, PythonAnywhere, Vercel
